# Steps to Run BhaiChara APP (Ride Sharing Application)

## Software to Install

1- Install XAMP
2- Install Android Studio
3- Install Eclipse JEE/IntelliJ Idea 
4- Install Postman

## Open The Project

1- Open the XAMMP 
	1.1- Start the Apache.
	1.2- Start the MySql.
	1.3- Open phpmyadmin in Browser.
	1.4- Import the Database thats On The Github.

2- Open the Android Studio (Android)
	3.1- Open the Android Site Code On Android Studio.

3- Open the Eclipse/IntelliJ (Server)
	2.1- Open the Server Site Code On the Eclipse.

4- For Admin Panel 
	4.1- Copy the Admin panel code 
	4.2- Pate in the xampp/htdocs/

## RUN The Code 

1- Android Studio 
	1.1- Connect the Phone
	1.2- Generate the Google Keys
	1.3- Replace them with the keys that I have used
	1.2- Run the Code 

2- Eclipse/IntelliJ 
	2.1- Run application as Spring Boot

 	(turn off the windows firewall to run the mobile app on local server)    

3- Admin Panel
	3.1- Open the Admin Panel On Browser
